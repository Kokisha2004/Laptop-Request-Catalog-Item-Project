#  Requirement Analysis Phase – Laptop Request Catalog Item

This folder contains the requirement analysis documents for the **Laptop Request Catalog Item** NM Project developed in **ServiceNow**.

---

##  Objective
To define the system requirements, user needs, and technology stack for automating the laptop request process.

---

##  Documents Included
- **Data Flow Diagrams (DFD):** Shows how data moves between the user, Service Catalog, and backend.  
- **User Stories:** Describes what the user and IT admin expect from the system.  
- **Solution Requirements:** Lists functional (catalog item, UI policy, workflow) and non-functional (reliability, performance) needs.  
- **Technology Stack:** ServiceNow platform using JavaScript, Service Catalog, UI Policy, and Update Sets.

---

##  Outcome
The requirement analysis defined what to build, how it works, and which technologies support it — forming the base for the design and development phases.
