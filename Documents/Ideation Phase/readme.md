#  Ideation Phase – Laptop Request Catalog Item

This folder contains the **Ideation Phase** documents for our NM project titled **“Laptop Request Catalog Item”**, developed using **ServiceNow**.

During this phase, our team focused on understanding the problem, identifying user pain points, and generating potential ideas for the solution.

---

###  Documents Included

1.**Brainstorming Template**  
   - Records creative ideas proposed by the team.  
   - Includes idea prioritization to select the most feasible and impactful solution.

2. **Define Problem Statement**  
   - Describes the core problem: manual laptop requests causing delays, errors, and inefficiency.  
   - Establishes the need for automation using ServiceNow’s Service Catalog.

3.  **Empathy Map Canvas**  
   - Helps visualize user experiences and challenges in the existing laptop request process.  
   - Captures user thoughts, needs, and frustrations to design a better workflow.

---

###  Outcome of Ideation Phase
Through brainstorming and analysis, the team concluded that creating a **Service Catalog Item in ServiceNow** would:
- Simplify the laptop request process,  
- Improve accuracy, and  
- Provide a dynamic, automated user experience.

This phase laid the foundation for the design and development steps in later stages of the project.
