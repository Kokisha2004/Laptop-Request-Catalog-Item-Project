
#  Project Planning Phase – Laptop Request Catalog Item

This folder contains the planning and scheduling materials for the **Laptop Request Catalog Item** NM Project developed using **ServiceNow**.

---

##  Objective
To plan the project timeline, deliverables, and team responsibilities before the implementation phase.  
This ensures systematic progress and on-time completion of all NM project stages.

---

##  Documents Included
1. **Project Plan / Timeline**
   - Outlines weekly milestones and deliverables for each phase.  
   - Includes ideation, design, requirement analysis, development, and testing stages.  

2. **Milestone Chart**
   - Displays key checkpoints such as catalog item creation, UI policy setup, and testing.  
   - Helps track project progress and ensure timely submissions.  

3. **Resource Allocation**
   - Defines each team member’s role and responsibility (e.g., documentation, configuration, testing).  
   - Ensures efficient teamwork and workload balance.  

---

##  Key Planning Highlights
- **Team Size:** 4 Members  
- **Platform:** ServiceNow Developer Instance  
- **Tools Used:** ServiceNow, Microsoft Word, Mural, Figma  
- **Timeline:** Planned completion across 5 phases – Ideation → Planning → Design → Requirement Analysis → Testing.  

---

##  Outcome
The planning phase provided a clear roadmap for the project execution, helping the team stay organized, meet deadlines, and maintain quality throughout all subsequent phases.
