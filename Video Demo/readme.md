#  Video Demonstration – Laptop Request Catalog Item

This folder contains the **video demonstration** of the NM Project **“Laptop Request Catalog Item”** developed using **ServiceNow**.

---

##  Purpose
To visually demonstrate how the **Laptop Request Catalog Item** works — from creation to testing — and to verify that all functionalities perform as expected.

---

##  Video Contents
The demo video covers the following steps:

1. **Update Set Creation**  
   - Creating and activating the update set “Laptop Request Project.”

2. **Catalog Item Setup**  
   - Creating the *Laptop Request* catalog item under the *Hardware* category.  
   - Adding variables: *Laptop Model*, *Justification*, *Additional Accessories*, *Accessories Details*.

3. **UI Policy Configuration**  
   - Showing how *Accessories Details* becomes visible and mandatory only when *Additional Accessories* is checked.

4. **UI Action (Reset Form)**  
   - Demonstrating the form reset functionality using the custom JavaScript action.

5. **Exporting and Retrieving Update Set**  
   - Exporting the update set from one instance and importing it into another.

6. **Testing the Catalog Item**  
   - Submitting a laptop request and validating dynamic field behavior and reset functionality.

---

##  Outcome
The video confirms that:
- The catalog item is fully functional.  
- Dynamic field visibility and reset actions work as expected.  
- Update set migration between instances is successful.  

##  Conclusion
This video demonstrates the complete working solution of the **Laptop Request Catalog Item**, showing how ServiceNow automates the laptop request process efficiently.
