# Laptop-Request-Catalog-Item-Project


# Project Overview

This project automates the process of requesting laptops within an organization using ServiceNow.
A custom Laptop Request Catalog Item was created to allow employees to easily request laptops with dynamic fields, UI policies, and reset functionality.
